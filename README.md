# phase1_dose_escalation_combination_therapy

In the command line, navigate to your project directory and run 

```
git clone https://github.com/swannyy14/phase1_dose_escalation_combination_therapy.git
```

or, alternatively, navigate to [this link](https://github.com/swannyy14/phase1_dose_escalation_combination_therapy) 
and manually download the repository as a ZIP file.

Open `use_guide.qmd` in R Studio for instruction on how to use the code for PO-TITE-CRM and simulation.

`R/ScenarioSimulation.R` contains the code used to run the simulations.
